# Match Animals to Environments

Small mobile-first single-page app. Drag animals to align with environments and press `Check`.

Files:
- `index.html` — main page
- `assets/main.js` — app logic
- `assets/styles.css` — styles
- `assets/mapping.json` — correct mapping

Deploy: push to GitHub and enable Pages from `main` branch (root). A GitHub Actions workflow is included.
